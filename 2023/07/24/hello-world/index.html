🎉 欢迎来到我的博客

大家好！这是我使用 Hexo 和 GitHub Pages 搭建的个人博客的第一篇文章。经过一番折腾，终于成功上线了！

🚀 关于 Hexo

Hexo 是一个快速、简洁且高效的博客框架。它有以下特点：

  • ⚡ 超快速度 - Node.js 所带来的超快生成速度
  • 📝 Markdown 支持 - 支持 GitHub Flavored Markdown 的所有功能
  • 🎨 丰富主题 - 拥有众多精美主题可供选择
  • 🔌 插件系统 - 强大的插件系统，轻松扩展功能
